DiffBlocks¶

class geneblocks.DiffBlocks(s1, s2, blocks)[source]¶

Class to generate and represent DiffBlocks.

Usage:

>>> DiffBlocks.from_sequences(s1, s2)
static from_sequences(s1, s2, blast_over=500, max_complexity=100000000.0)[source]¶

Create DiffBlocks by comparing two sequences

Parameters

s1, s2

Two sequences, either “ATGC…” strings or Biopython records.

blast_over

A blast will be triggered to accelerate homology finding if len(s1) + len(s2) > blast_over.

max_complexity

If len(s1) * len(s2) is over max_complexity, no analysis is done and s1 is just labeled as a “change” of s2 (useful internally during the recursions of this method)
